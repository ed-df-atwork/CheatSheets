- [05 CTEs and Subqueries](05_ctes_and_subqueries.md) - Common Table Expressions and nested queries.

- [10 Reporting Tools](10_reporting_tools.md) - SQL for BI tools and dashboarding.
- [11 Stored Procedures](11_stored_procedures.md) - Scripting, Variables, and Logic.
- [12 Data Profiling](12_data_profiling.md) - Auditing data quality and distributions.
- [13 Performance Tuning](13_performance_tuning.md) - Execution plans and query optimization.

Use these files as a quick reference during development or interview preparation. Most examples follow standard SQL syntax (ANSI SQL).
